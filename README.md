## Descrição
Nosso projeto tem a proposta de criar um aplicativo de controle financeiro para aqueles que sofrem muitas vezes com falta de controle dos seus gastos. Oferecendo a descrição e listagem de seus gastos, os ordenando por um sistema de prioridade para assim ajudar o usuário a organizar suas prioridades.
